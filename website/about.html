Project context

Public documentation for an evolving identity platform.

MrWhoOidc is focused on delivering a standards-based identity provider with enough surrounding operational tooling to be usable in real environments, not just in a protocol demo.

The public repository exists to publish deployment guidance, integration examples, and public-facing documentation. It complements the product source tree by giving operators and developers a simpler entry point.
